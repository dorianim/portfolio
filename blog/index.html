Handling forms in Hugo

November 26, 2021

A while ago, I migrated the website of Itsblue from Grav to Hugo. The main reason for doing so was that I wanted clean version control, local testing and a static site. But this came with a problem: In Grav, I could use their built-in form engine to process my contact form. In Hugo on the other hand, as there is no backend, processing forms is not as straight forward, Hugo just creates static HTML files.
